Implementation: Nearest Neighbour with 2-opt optimization in C++

Specifications:
Input: text file.

Each line defines a city and has 3 numbers separated by space.
The first number is the city id.
The second number is the city's x coordinate.
The third number is the city's y cooridnate.

Output: Text file (Input_filename.tour)
First line contains tour length

List of cities visited in order. Each consecutive line is the city visited.
